## 关于 净化页面 功能

- 减少```大量的```置顶遮挡物。
- 减少```大量的```多余元素。
- 减少```频繁地```手动关闭遮挡物。
- 让页面焕然一新，提升体验感。

## 关于 展示全屏 功能

- 拓展纵向视野。
- 拓宽横向视野。
- 减少```频繁地```上下滚动鼠标。
- 减少```频繁地```左右滚动鼠标。
- 释放鼠标手，提升体验感。
